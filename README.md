# txtar とは

- 複数ファイルを1テキストで表現
- mimeマルチパートより遥かに簡単
- ディレクトリ階層も扱える
- 行頭に "-- " を含むファイルは扱えない？
- テストfixtureとか書くのとかに便利そう
